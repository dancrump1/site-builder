# Craft CMS Dev Environment
Scaffolding for a CraftCMS 4 project powered by Vite and setup with DDEV

## Requirements
-   Docker, https://www.docker.com
-   DDEV, https://ddev.com

## Steps
``git clone``
``ddev config``
``make install``
``Delete the composer lock file``
``composer install``
``make install``
``ddev start``
``make dev``
